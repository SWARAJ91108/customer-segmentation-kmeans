# Customer Segmentation using K-Means Clustering

## 📌 Overview
This project segments customers based on Annual Income and Spending Score using K-Means Clustering.

## 🎯 Problem Statement
Retail businesses often send the same promotional offers to all customers. This project groups customers into meaningful segments to enable personalized marketing.

## 📂 Dataset
- Mall_Customers.csv
- Source: Kaggle

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Methods Used
- K-Means Clustering
- Elbow Method
- Silhouette Analysis

## 📈 Results
- Optimal K = 5
- Successfully identified customer segments for targeted marketing
